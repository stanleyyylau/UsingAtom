## Atom 基础功能说明

### command + P
快速寻找 -- 快速跳到你想要的档案
目录名 文件名   可以找到那个目录下的文件
source index 可以打开source 文件夹下的index
.git
.publish
之类灰色的档案是搜索不到的

### command + 1 AND  command + 2  AND  command + n
迅速切换tab

更实用
view -- split -- right 可以切分视图成左右两边

### command + n
开启一个新的档案


### 一直按command + D
选一个HTML tag 然后按command D 就可以选到一对
一直按command + D 可以一直选重复的字段 eg: class

### command + l
可以选择一整行

### command + control + m
选中一块内容
